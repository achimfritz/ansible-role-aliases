Ansible aliases Role
====================

An ansible role for managing mail aliases

Role Variables
--------------

    aliases_list: a list of dictionaries holding the user and the alias
        - user: postmaster
          alias: root

Example Playbook
-------------------------

    - hosts: servers
      roles:
         - { role: MichaelRigart.aliases }
